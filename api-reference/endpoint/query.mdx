### Query
<ParamField query="proj_path" type="string">
   This is project id for your Berri App. It's provided in the response after creating your app
   Example: indexes/ishaan@berri.ai/6663d98e-f3e2-42b9-a79e-2d92199b85fa
</ParamField>

<ParamField query="model" type="string" optional>
  There are currently 2 types of models Berri supports
  Default: Uses text-davinci-003
  1. **gpt-3.5-turbo**: Uses chatGPT turbo api
  2. **text-davinci-003**: uses the GPT-3 model 
</ParamField>

<RequestExample>
```python python
import requests

url = "https://shareddbstorequery-7bea-8hjw.zeet-berri.zeet.app/berri_query"

querystring = {"proj_path": "indexes/ishaan@berri.ai/6663d98e-f3e2-42b9-a79e-2d92199b85fa",
                "query": "who is ishaan",
                "model": "gpt-3.5-turbo"}

response = requests.post(url, params=querystring)

print(response.text)
```
</RequestExample>
