---
title: "Website / File Upload"
api: "POST https://sharedinfraingestserver-qic2.zeet-berri.zeet.app/berri_pipeline"
contentType: "form-data"
description: "This endpoint create a chatGPT app for a single website or file"
---

### Query

<ParamField query="user_email" type="string">
  Pass in your work / school email.
</ParamField>

<ParamField query="input_url" type="string" optional>
  Pass in the website url you are trying to extract responses from (e.g. https://mintlify.com/docs/components/api/param)
</ParamField>

<ParamField query="api_url" type="string" optional>
  Pass in the api url you are trying to extract responses from (e.g. https://api.semanticscholar.org/graph/v1/paper/search?query=covid%26year=2020-2023%26openAccessPdf%26fieldsOfStudy=Physics,Philosophy%26fields=title,year,authors)
</ParamField>

<ParamField body="file" type="form-data">
  Upload the file (.docx, .pdf, .pptx) (e.g. Berri_Financial_Statement.pdf)
</ParamField>


### Response

<ResponseField name="api_endpoint" type="string">
  This will return an api endpoint which you can use to query your app.
</ResponseField>

<ResponseField name="website_endpoint" type="string">
  This will return a website endpoint which you can use for quick prototyping.  
</ResponseField>

A sample request for creating a Berri app from a single file upload (a pdf in this case).
<RequestExample>

```bash Single File Upload Request
curl -X POST \
  'https://dataingestserver.ishaan-jaff.repl.co/berri_pipeline?user_email={user_email}' \
  -H 'content-type: multipart/form-data; boundary=----WebKitFormBoundary7MA4YWxkTrZu0gW' \
  -F 'file=Berri_Financial_Statement.pdf'
```

</RequestExample>

A sample request for creating a Berri app from an API url
<RequestExample>
```bash API Url Request
curl -X POST \
  'https://dataingestserver.ishaan-jaff.repl.co/berri_pipeline?user_email={user_email}&api_url=https://api.semanticscholar.org/graph/v1/paper/search?query=covid%26year=2020-2023%26openAccessPdf%26fieldsOfStudy=Physics,Philosophy%26fields=title,year,authors' \
```
</RequestExample>

A sample request for creating a Berri app from a website url
<RequestExample>
```bash Website Url Request 
curl -X POST \
  'https://dataingestserver.ishaan-jaff.repl.co/berri_pipeline?user_email={user_email}&input_url=https://www.tryfondo.com/guides/founders-guide-delaware-franchise-tax' \
  -H 'Postman-Token: 52871bf0-265d-4b5b-9c78-df31d81f7a95' \
  -H 'cache-control: no-cache'
```
</RequestExample>
